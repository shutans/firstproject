How to Define & Style Fonts with CSS

Tag Name Description
font-family Font Family Defines the font family that will be used on the page/element.
@font-face Font Face rule Allows external fonts or font files to be imported directly into your project.
font-weight Font Weight Defines the thickness of the font.
font-size Font Size Defines the size of the font.
font-style Font Style Defines the style of the font.
line-height Line Height Declares the vertical spacing between lines of text.

Best Practices:

  • Use fallback fonts, e.g. font-family: Verdana, Arial, sans-serif;
  • Surround font names with quotation marks if they are multiple words, e.g. font-family: 'Times New Roman';
